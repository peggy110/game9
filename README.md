# 貪吃蛇遊戲

這是一個用 HTML5 Canvas 製作的經典貪吃蛇網頁遊戲。

## 操作方式
- 使用鍵盤方向鍵或 WASD 控制蛇的移動。
- 吃到食物會加分並變長。
- 撞到牆壁或自己會遊戲結束。
- 點擊「重新開始」按鈕可重新遊玩。

## 遊戲特色
- 現代美化介面，漸層背景、圓角、陰影。
- 支援電腦瀏覽器即玩。

## 如何開始
直接打開 [`drink-game.html`](./drink-game.html) 即可遊玩。
